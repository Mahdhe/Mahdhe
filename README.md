> Full-stack salon management system with Angular frontend and Spring Boot REST API backend
- 📅 Appointment booking & management with staff assignment
- 👥 Role-based access (Admin & Staff) with Spring Security + JWT
- 💰 Revenue analytics dashboard and payment processing
- 📬 Automated email notifications via JavaMail (SMTP)
- **Stack:** Java · Spring Boot · Angular · MySQL · Spring Security · Maven

---

#### 💼 [EmployNet — Job Portal System](https://github.com/Mahdhe/employnet-jobportal)
> Full-stack web-based job portal connecting Job Seekers, Employers, and Admins
- 🗺️ Geolocation-based job matching with Haversine formula + Leaflet.js interactive map
- 🤖 AI-powered match analysis scoring applicants by education level and ranking
- 💬 Real-time chat/inquiry system between job seekers and employers
- 🔐 Secure authentication with bcrypt, CSRF protection, and PDO prepared statements
- **Stack:** PHP · MySQL · Bootstrap · jQuery · Ajax · PHPMailer · Leaflet.js
